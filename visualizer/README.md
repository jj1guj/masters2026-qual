# Periodic Patrol Automata - ビジュアライザ

第三回マスターズ選手権-予選- (Periodic Patrol Automata) 用のWebビジュアライザです。

## 起動方法

```bash
./visualizer/run.sh
```

ビルド後、http://localhost:8088 でブラウザからアクセスできます。

## 機能

### 入力生成
- シード値と問題タイプ (A/B/C) を指定して入力データを自動生成
- シード値や問題タイプの変更時に即座に反映

### 入出力
- 入力データ・出力データのテキスト貼り付け
- ファイル読み込みボタン

### ソルバー実行
- ボタン一つで `solver/` のビルドと実行が可能
- 実行結果は出力データ欄に自動で反映

### グリッド表示
- 20×20 グリッドの壁配置をCanvas上に描画
- 元の壁（赤）と追加壁（青）を色分け表示
- 警備対象マス（緑）の表示
- ロボット初期位置と方向の表示
- 巡回経路の表示
- 周期前経路の表示切替

### アニメーション
- 再生 / 一時停止 / ステップ送り / ステップ戻し
- 速度調整（1〜60ステップ/秒）
- タイムラインスライダーで任意のステップにジャンプ
- ロボットの現在位置と直近の軌跡をリアルタイム表示

### スコア計算
- 入出力データからスコアを自動計算
- N, A_K, A_M, A_W, K, M, W, V, Score を情報バーに表示

## 技術構成

- **バックエンド**: Rust (actix-web 4)
- **フロントエンド**: HTML5 Canvas + JavaScript (ライブラリ不使用)
- **ポート**: 8088
